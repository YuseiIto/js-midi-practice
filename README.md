[動画](https://x.com/yuseiito_/status/1807261800293093681)


## Development

1. Nixでdevshellを立ち上げる。

```zsh
make devshell
```

2. npmしぐさ


```zsh
npm install
```

3. ビルドして実行

```zsh
make build
make run
```

OR simpy

```zsh
make dev
```
